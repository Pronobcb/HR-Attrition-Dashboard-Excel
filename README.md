# 📊 HR Attrition Dashboard

## 📌 Project Overview
This project presents an interactive HR Attrition Dashboard that analyzes employee turnover and workforce trends. It helps identify key factors influencing attrition and supports data-driven decision-making for improving employee retention.

## 🎯 Objectives
- Analyze overall attrition trends  
- Identify high attrition departments and job roles  
- Understand the impact of age, education, and marital status  
- Provide insights for better HR decision-making  
---
## 📊 Key Metrics
- Total Employees: 1470  
- Attrition Count: 237  
- Active Employees: 1233  
- Attrition Rate: 16.12%  
- Average Age: 37

## 🔍 Key Insights
- Sales and R&D departments have higher attrition  
- Employees aged 25–34 show the highest turnover  
- Bachelor's degree holders have higher attrition  
- Single employees tend to leave more  
- Some job roles have significantly higher attrition
